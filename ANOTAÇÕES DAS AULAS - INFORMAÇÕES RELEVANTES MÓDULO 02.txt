
EMOÇÕES DE CORES 

NO VISUAL CODE É POSSÍVEL USAR TRANSPARÊNCIA NAS CORES - NAS BARRAS AO LADO

HARMONIA DE CORES - CORES PRIMÁRIAS (AMARELO, VERMELHO, AZUL); CORES SECUNDÁRIAS (LARANJA, VIOLETA, VERDE); CORES TERCIÁRIAS (TODAS AS OUTRAS QUE SÃO MISTURAS ENTRE AS PRIMÁRIAS E SECUNDÁRIAS - PRIMEIRO NOME DA COR PRIMÁRIA E DEPOIS O NOME DA SECUNDÁRIA)

CIRCULO CROMÁTICO - IMPORTANTE SEMPRE VERIFICAR PARA TER BASE NAS CORES

TEMPERATURA DE CORES - CORES FRIAS E CORES QUENTES

CLASSIFICAÇÕES BUSCANDO HARMONIA DE CORES (PALETA DE CORES - 3 A 5 CORES)

CORES COMPLEMENTARES - CORES QUE MAIS DÃO CONTRASTES ENTRE SI (CORES OPOSTAS NO CIRCULO CROMÁTICO)

CORES ANÁLOGAS - CORES QUE NÃO TEM MUITO CONTRASTE ENTRE ELAS, MAS QUE SÃO PERCEPTÍVEIS (HARMONIA ENTRE CORES)

CORES ANÁLOGAS RELACIONADAS - PEGA A COR AO LADO, PULA OUTRA E PEGA MAIS UMA COR (PARA QUALQUER LADO)

CORES INTERCALADAS - PULA UMA E PEGA A OUTRA (PARA QUALQUER LADO)

CORES TRIÁDICAS - PEGA UMA COR DE REFERÊNCIA E PULA 3 CORES, PEGANDO A PRÓXIMA COR (PARA QUALQUER LADO)

CORES EM QUADRADOS - PEGA UMA COR DE REFERÊNCIA E PULA DUAS CORES, PEGANDO A PRÓXIMA COR (PARA QUALQUER LADO)

CORES TETRÁDICAS - FORMA UM RETÂNGULO UTILIZANDO CORES COMPLEMENTARES

MONOCROMIA - TRABALHA SOMENTE COM UMA COR

SELETORES PERSONALIZADOS

class - é para múltiplos elementos - representado pelo .

pseudoclasses - representado pelo :

pseudoelementos - mexem no conteúdo periférico do elemento - representado pelo ::

Children - representado por >


MODELO DE CAIXAS (BOX MODEL)

Aninhamento - uma caixa dentro da outra (sucessivas vezes)

ALTURA - height; LARGURA - width; LINHA EM VOLTA DO CONTEUDO - border; ESPAÇO EM VOLTA DA LINHA DO CONTEÚDO - padding; ESPAÇO EXTERNO - margin; ESPAÇO FORA DA BORDAR (CONTRONO) - outline

TIPOS DE CAIXA - box-level (sempre se inicia em uma linha nova; ocupa sempre a tela inteira) - <div>; inline-level (continuará na mesma linha; não ocupa a largura inteira da tela) <span>;

TESTAR O BOX LEVEL E INLINE BOX NO DEVTOOLS DO PRÓPRIO GOOGLE - VERIFICAR TIPOS DE MARGENS

PARA CENTRALIZAR O BOX DEVE SER UTILIZADO O COMANDO auto
