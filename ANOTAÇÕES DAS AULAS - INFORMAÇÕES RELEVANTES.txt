"EU DESENVOLVO EM HTML (HYPERTEXT MARKUP LANGUAGE) E CSS (CASCADING STYLE SHEETS)" - LINGUAGEM

HTML É UMA LINGUAGEM FOCADA EM CONTEÚDO (TEXTOS, IMAGENS, VÍDEOS, TABELA, LISTAS...)

CSS É FOCADO EM DESIGN (CORES, SOMBRAS, TAMANHOS, POSICIONAMENTO...)

JAVASCRIPT - LINGUAGEM DE PROGRAMAÇÃO - É FOCADO EM INTERAÇÕES (MENUS INTERATIVOS, ANIMAÇÕES, POPUPS, VALIDAÇÕES...)

CONTEÚDO EM HTML - TAGS (<>) - EXEMPLO DE TÍTULO <h1> (abertura de tag); </h1> (fechamento de tag) - EXEMPLO DE PARAGRAFO <p> (abertura de tag); </p> (fechamento de tag) - <img scr= (foto.png); alt= (exemplo de foto)

ESTILO EM CSS - h1 { (seletor) }    (toda declaração em CSS deve ter ; no final)
